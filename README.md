# Apple Sales Analysis using PostgreSQL

## Project Overview

This project analyzes Apple retail sales data using PostgreSQL and pgAdmin.

The database contains over 1 million sales transactions across multiple stores and product categories.

## Database Schema

Tables:

- category
- products
- stores
- sales
- warranty

## Technologies Used

- PostgreSQL
- pgAdmin 4
- SQL
- GitHub

## Key SQL Concepts

- Joins
- Aggregations
- CTEs
- Window Functions
- Indexing
- Query Optimization

## Business Questions Solved

1. Number of Stores Per Country
2. Total Units Sold By Each Store
3. Total Transactions in December 2023
4. Stores That Never Received Warranty Claims
5. Percentage of Warranty Void Claims
6. Highest Selling Store
7. Unique Products Sold
8. Average Product Price Per Category
9. Warranty Claims Filed in 2020
10. Best-Selling Weekday Per Store

## Dataset

The project uses a retail sales database containing 1M+ transaction records.
